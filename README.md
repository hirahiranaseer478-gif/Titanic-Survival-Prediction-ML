
# Titanic Survival Prediction (ML)

A Machine Learning project implemented in Jupyter Notebook that predicts passenger survival on the Titanic using classification algorithms.

## 📊 Results
After an 80/20 train-test split, the models achieved the following accuracy:

| Model | Accuracy |
| :--- | :---: |
| **Logistic Regression** | **81.01%** |
| **Decision Tree Classifier** | **80.45%** |
| **Naive Bayes (Gaussian NB)** | **75.42%** |

## 🛠️ Tech Stack
* Python (Jupyter Notebook)
* Pandas & Scikit-Learn
